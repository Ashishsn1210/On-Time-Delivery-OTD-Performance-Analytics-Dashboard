# On-Time-Delivery-Performance-Analytics-Dashboard

## Project Overview

This project presents an On-Time Delivery (OTD) Performance Dashboard developed using Google Looker Studio to monitor delivery performance and order completion efficiency. The dashboard tracks delivery commitments against actual dispatch performance to help business and sales teams evaluate how effectively customer orders are fulfilled on time. The dashboard provides a centralized view of order commitments, dispatch quantities, and delivery adherence metrics, enabling teams to quickly identify delivery gaps and improve planning.


## Business Problem
Before implementing this dashboard, delivery performance tracking was primarily done through manual spreadsheet reports and fragmented operational data.

This created several operational challenges:
- Delivery performance metrics were difficult to track consistently.
- Identifying delays required manual analysis of multiple reports.
- Sales and operations teams lacked a quick overview of delivery commitments versus actual dispatch.
- Review meetings often spent significant time compiling data rather than analyzing performance.

Without a centralized reporting system, it was difficult to answer key operational questions such as:
- How many orders were committed for dispatch?
- How many were actually dispatched on time?
- What percentage of orders met delivery commitments?
- How is delivery performance trending over time?

These challenges created the need for a structured analytics dashboard that could monitor delivery performance in real time.

## Solution Overview
To address these challenges, an On-Time Delivery Performance Dashboard was developed using Looker Studio. The dashboard consolidates operational order and dispatch data into a single visual interface, allowing teams to monitor delivery performance efficiently.

Key capabilities of the dashboard include:
- Tracking total purchase order quantities
- Monitoring dispatch quantities
- Measuring On-Time Delivery percentage
- Measuring Delivery Adherence percentage
- Visualizing order completion trends over time

## Business Impact
The implementation of the OTD dashboard significantly improved delivery performance visibility within the organization.
Before the dashboard was implemented, delivery analysis required manual data compilation and interpretation.
With the dashboard in place, teams are able to quickly evaluate delivery performance during operational review meetings.

Key Improvements
- Delivery performance metrics became easily accessible through a centralized dashboard
- Sales and operations teams gained visibility into order completion and dispatch trends
- Leadership could quickly evaluate delivery performance during weekly reviews
- Operational discussions shifted from data compilation to performance improvement discussions

## Key Metrics Tracked
The dashboard monitors several key operational metrics used to evaluate delivery performance.

#### Purchase Order Quantity (PO Qty): Represents the total number of orders committed for delivery within the selected time range.
#### Dispatch Quantity: Represents the total quantity of orders that were successfully dispatched.
#### On-Time Delivery (OTD): On-Time Delivery measures the percentage of orders delivered within the committed schedule.
#### Delivery Adherence (DA%): Delivery Adherence measures how closely actual dispatches align with planned delivery commitments.

## Dashboard Features
The dashboard provides several interactive capabilities to support operational analysis.
#### Date Range Filtering: Users can filter delivery performance for a specific time period to analyze trends and review recent operational performance.
#### Customer-Level Analysis: Delivery performance can be filtered by customer to evaluate delivery reliability across different accounts.
#### Delivery Performance Trend: The dashboard visualizes delivery commitments, order completion counts, and OTD performance across time.

## Tools Used
